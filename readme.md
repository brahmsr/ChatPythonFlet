# Commands

1- Install venv: 
```
python -m venv .venv
```

2- start venv:
```
.venv/scripts/activate
```

3- Install dependencies:
```
pip i -r requirements.txt
```

4- create migration
```
py manage.py makemigrations
```

5- run migrations:
```
py manage.py migrate
```

6- run backend:
```
py manage.py runserver
```

7- run front: 
```
flet run app
```
